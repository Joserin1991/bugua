```

```

## 命名与交付规则
- 全部小写连字符命名，PNG 透明底（除注明 JPG）
- 透明底务必勾选"透明背景/去背"；若生成带灰底，发我用亮度转透明管线处理
- 同一项生成 4 张挑 1 张，挑选标准见各项"验收要点"
