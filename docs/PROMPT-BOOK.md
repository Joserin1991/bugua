## 命名规则（全局强制）

1. 目录：public/brand/ ·bg/ ·sym/ ·ui/ ·icon/ ·fx/ ·char/ ·motion/ ·page/（按包入目录）
2. 文件名：全小写英文+连字符+两位序号（如 enso-03.png）；多态后缀 -on/-off；生肖等语义名（zodiac-dragon.png）
3. 格式：图片一律 PNG-24 透明底（纸纹平铺 JPG 85）；动画交 MP4/序列帧由我转制；组件为代码资产无文件
4. 尺寸：按条目标注；工具只有固定档位时选更大档，由我裁切
5. 自检：无彩色（除注明朱砂）/ 无文字水印 / 背景透明 / 笔触有飞白，任一不满足重新生成
